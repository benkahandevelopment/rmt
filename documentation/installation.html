Installation

Please note that as of January 2018 you can only install this package manually - the packaged Chrome extension will be uploaded to the Web Store soon.

Manual Installation

  1. Download the full package from GitHub
  2. Add the download folder as an unpacked extension in Developer Mode on the Chrome Extensions page by visiting chrome://extensions

You will know the installation is successful because you'll see the extension at the top of the extensions list, as well as a new icon to the right of the URL bar.
