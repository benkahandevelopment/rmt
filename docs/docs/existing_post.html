Modifying an Existing Post with rMT

  1. Ensure the active tab is on a Match Thread post you have already created
  2. Click on the rMT icon to the right of the Chrome URL bar:
  3. Input/check that all the meta, teams, stats and updates are correct
  4. Click Save & Update Post

That's it! The tool will automatically remove old data from your post, replace it with the data you've provided and format it using reddit markdown before hitting the "save changes" button for you. It may take a second or two but you should see your changes almost immediately.
