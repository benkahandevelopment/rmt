Macro Appendix

Overview

Add macros to your post or theme to add dynamic values from the Popup to your post.

For example, if your home team is "Spurs", you can add the macro {{META_HOME}}, which will be replaced by [](#rmt-start-meta-home)Spurs[](#rmt-end-meta-home). This will output Spurs.

Macros
