Teams

Overview

The "Teams" tab you with the tools to set lineups/benches for home and away teams, as well as any applicable icons to signify player events. These icons are:

  • Goal
  • Own Goal
  • Booking
  • Sending Off
  • Injury
  • Sub Off
  • Sub On
  • MoTM

Adding/Changing Icons

Pressing the plus symbol to the left of any player field will bring up the icon edit modal. From here you can add a new icon and remove existing ones.

When you click "Confirm Changes", whatever icon is in the "New Event" field is added to the player. To just remove an icon, ensure that "No Icon" is selected.

To delete an icon, click the delete icon .
